## 🧩 What is Agent-Debate-Consensus?

Agent-Debate-Consensus is a software platform where multiple AI agents discuss any topic in real time. You act as the moderator and watch the conversation unfold. This tool uses several language models simultaneously and shows their views side by side. The platform runs smoothly on Windows.

It is built using web technologies like Next.js and TypeScript. The interface uses Tailwind CSS for styling, and Zustand helps manage the app’s state. The AI power comes from OpenRouter, which connects multiple large language models. You don’t need to know programming to run it—just follow the steps below.
