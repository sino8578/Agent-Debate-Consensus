## Чому це змінює гру?

Одна AI-модель — одна точка зору. Одна форма упередження. Один спосіб міркування.

**Agent Debate Consensus** дає вам чотири найсильніших моделі одночасно, в реальному часі, змаганням одна з одною.

Різні моделі мають різні дані тренування, різні упередження, різні стилі мислення. Коли ви змушуєте їх дебатувати тему й захищати позиції перед опонентами, ви отримуєте набагато повнішу картину, ніж могла б дати будь-яка окрема модель. Суперечності виявляються. Логічні діри викриваються. Консенсус, коли він виникає, справедливо заробляється через аргументацію.

**Призначений для:**
- Перевірки важливих рішень з кількох перспектив
- Дослідження суперечливих тем без однобічності
- Отримання сформованої, колективної відповіді замість однієї думки
- Навчання розуміння того, як думають різні AI-системи
- Документування консенсусу для вашої команди

## Повний набір можливостей

✨ **Мультиагентні дебати** — До 8 AI-агентів дискутують одночасно (налаштовується через `MAX_ACTIVE_MODELS`). Кожна модель видає незалежну позицію, потім реагує на інших у наступних раундах.

⚡ **Потокова передача в реальному часі** — Токен за токеном через Server-Sent Events. Жодного очікування на повну відповідь — спостерігайте, як моделі видають думки palavra за словом, живо й динамічно.

📌 **Система автодоповнення @згадок** — Напишіть `@`, щоб побачити випадаючий список активних агентів. Навігація клавішами (стрілки, Enter). Згадки автоматично підсвічуються кольором моделі.

🎭 **AI-модератор зі зірочкою** — Призначте будь-яку модель модератором (позначена зірочкою ★). Модератор синтезує аргументи, підсумовує консенсус, поєднує позиції. Автоматичне переназначення при критичному збої.

🧠 **Розумний двигун дебатів** — Відстеження раундів per-модель (кожна модель говорить 2 раунди), черга з пріоритетами та кулдаунами. Не дозволяє агентам говорити одночасно. Справедливість гарантована.

🔌 **Інтелектуальне управління контекстом** — Перше питання користувача завжди закріплене вгорі. Правильне відображення ролей агентів. Контекст не втрачається під час довгих дебатів.

🎨 **3 режими температури** — Creative (0.9) для вільних ідей, Balanced (0.7) для змішаного стилю, Precise (0.3) для логічної строгості. Вибір залежить від потреби.

📌 **Закріплена тема як смуга** — Перше повідомлення користувача завжди видно вгорі як смуга контексту. Тема ніколи не теряється в дискусії.

💾 **Сесії дебатів зі збереженням** — Збереження, завантаження, видалення дискусій. Автоматичне збереження при перемиканні між вкладками. Повернення до цікавих дебатів в один клік.

⚡ **Кнопка розвинення (Boost)** — Блискавка на повідомленні для запиту до моделі розвинути ідею глибше. Розгляд аспектів, які пропустили.

🔍 **Каталог агентів з 400+ моделями** — Пошук, фільтрування й додавання моделей з OpenRouter на лету. Новітні моделі, екзотичні провайдери, спеціалізовані моделі.

🧩 **Розширене мислення (Chain-of-Thought)** — Повна підтримка моделей із внутрішнім міркуванням. Секція «Thinking...» згортувана для чистоти. Розгорніть, щоб побачити внутрішню логіку.

🚨 **Обробка та індикація збоїв** — Червоні індикатори помилок на повідомленнях. Тултіп з деталями помилки. Автоматичне переназначення модератора при критичному збої моделі.

📥 **Експорт дебатів в Markdown** — Завантажте всю дискусію як .md файл. Поділіться з командою, документуйте рішення, архівуйте для майбутнього.

📋 **Копіювання повідомлень** — Копіюйте будь-яке повідомлення в Markdown форматі в буфер обміну з одним кліком.

📱 **Мобільна адаптивність** — Сайдбар накладається на вміст на мобільних пристроях. Повна функціональність на смартфонах, планшетах, десктопах.

🎚️ **Слайдер розміру шрифту** — Персоналізуйте розмір тексту без перезавантаження. Доступність для всіх.

🌓 **Темна/світла тема** — Перемикач у шапці. Premium dark UI для тривалих сесій. Light режим для денного часу.

⬇️ **Автоколапс довгих повідомлень** — Повідомлення понад 500 символів згортуються. Розгорніть натисненням для читання. Чистота й контроль.

🔐 **Публічний режим** — Деплой без вашого ключа. Користувачі вводять свій OpenRouter API ключ. Ідеально для демо, громадських сервісів, навчальних платформ.

⏳ **Ліміт агентів з лічильником** — `MAX_ACTIVE_MODELS` обмежує одночасних агентів (за замовчуванням 8). Лічильник в UI показує використання.

## Як працює алгоритм дебатів

**Фаза 1: Ін'єкція теми**
Ви вводите питання або тему. Усі активні агенти отримують його як контекст.

**Фаза 2: Незалежні позиції (Раунд 1)**
Кожен агент формулює самостійну позицію, не знаючи про інших. Паралельна обробка гарантує справедливість.

**Фаза 3: Крос-запилення (Раунди 2+)**
Кожен агент тепер бачить відповіді інших. Генерується справжня реакція, контраргументи, визнання гарних точок.

**Фаза 4: Модерація (ваша роль)**
Ви втручаєтесь будь-коли:
- Уточнюйте терміни, якщо виникає плутанина
- Оскаржуйте слабкі аргументи з доказами
- Попросіть конкретного агента захистити позицію
- Модератор синтезує дебати й підсумовує консенсус

**Фаза 5: Конвергенція**
Через ітеративний обмін агенти:
- Уточнюють позиції на основі критики
- Визнають сильніші аргументи й коригують точку зору
- Рухаються до консенсусу (або виявляють непримиренні розходження, які варто знати)

## Швидкий старт за 5 хвилин

**1. Клонування репозиторію**

```bash
git clone https://github.com/Lexus2016/Agent-Debate-Consensus.git
cd Agent-Debate-Consensus
```

**2. Встановлення залежностей**

```bash
npm install
```

**3. Налаштування змінних оточення**

Створіть файл `.env.local` у кореневій папці проекту:

```env
# Для публічного режиму — залиште порожнім й дайте користувачам вводити свої ключі
OPENROUTER_API_KEY=

# Або для власного ключа — вставте ваш ключ OpenRouter
OPENROUTER_API_KEY=sk_your_openrouter_key_here

# Максимум одночасних агентів (за замовчуванням 8)
MAX_ACTIVE_MODELS=8
```

Отримайте безплатний API-ключ на [openrouter.ai/keys](https://openrouter.ai/keys).

**4. Запуск сервера розробки**

```bash
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) в браузері.

**Готово!** Виберіть агентів з сайдбара й введіть перше питання.

## Як користуватися платформою

### Початок дебатів

1. **Виберіть агентів** на лівій панелі. Кліком активуйте модельки, які хочете залучити до дискусії.
2. **Введіть питання або тему** в поле введення знизу. Це стане закріпленим контекстом для всіх агентів.
3. **Спостерігайте в реальному часі** — агенти видають свої позиції потоком, токен за токеном.

### Управління дебатами

- **@згадки для спрямування** — Напишіть `@Claude` чи `@Gemini` для адресації питання конкретній моделі.
- **Встановіть модератора** (натиснете зірочку ★ на агенту) — вибраний агент синтезує дебати й підсумовує позиції.
- **Втручайтеся будь-коли** — додавайте уточнення, оскаржуйте аргументи, попросіть розвинути ідею.
- **Кнопка розвинення** (⚡) — попросіть агента більш детально розглянути питання.

### Контроль розмови

- **Температура** — Creative (вільна творчість), Balanced (змішано), Precise (логіка).
- **Розмір шрифту** — Слайдер для комфорту читання.
- **Темна/світла тема** — Перемикач у шапці.
- **Копіювання** — Копіюйте повідомлення в Markdown формату.

### Збереження й експорт

- **Автозбереження** — Сесія зберігається в localStorage при перемиканні.
- **Експорт в Markdown** — Завантажте всю дискусію як .md файл для команди.
- **Перезавантаження** — Повернитеся до збереженої сесії з історією.

### API кінцеві точки

**POST `/api/chat`** — Stream SSE дебати
```
Request:
{
  "message": "Яка найкраща мова програмування?",
  "models": ["@Claude", "@Gemini", "@Grok"],
  "temperature": 0.7,
  "moderator": "@Claude"
}

Response: Server-Sent Events (токени в реальному часі)
```

**GET `/api/models`** — Каталог 400+ моделей
```
Response:
{
  "models": [
    { "id": "claude-3-haiku", "name": "Claude Haiku 4.5", "provider": "Anthropic" },
    ...
  ]
}
```

## Часті запитання

**Чи мій API-ключ безпечний?**
Ключ передається прямо від браузера до OpenRouter (публічний режим) або від сервера. Не реєструється, не логується.

**Скільки коштує запуск 8 агентів одночасно?**
Залежить від вибраних моделей. Kimi K2 та Grok 4.1 дешеві (~$0.001/1K токенів). Claude й Gemini дороговіші. OpenRouter розраховує по спожитих токенах.

**Можу я додати свою модель?**
Так. Будь-яка модель з OpenRouter доступна в каталозі. Новые моделі синхронізуються автоматично.

**Як змінити тему дебатів?**
Почніть новий чат (нова сесія). Перше повідомлення стає закріпленою темою для всієї дискусії.

**Що робити, якщо модератор дасть помилку?**
Система автоматично переназначає іншого агента модератором. Дебати продовжуються без перерви.

**Розміщення на власному сервері?**
Повна support. Деплой на Vercel, Netlify, власний сервер — будь-де. Вам потрібні тільки Node.js 16+ і OpenRouter ключ.
