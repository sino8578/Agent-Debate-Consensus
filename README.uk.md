## Чому це змінює гру?

Одна AI-модель — одна точка зору. Одна форма упередження. Один спосіб міркування.

**Agent Debate Consensus** дає вам чотири найсильніших моделі одночасно, в реальному часі, змаганням одна з одною.

Різні моделі мають різні дані тренування, різні упередження, різні стилі мислення. Коли ви змушуєте їх дебатувати тему й захищати позиції перед опонентами, ви отримуєте набагато повнішу картину, ніж могла б дати будь-яка окрема модель. Суперечності виявляються. Логічні діри викриваються. Консенсус, коли він виникає, справедливо заробляється через аргументацію.

**Призначений для:**
- Перевірки важливих рішень з кількох перспектив
- Дослідження суперечливих тем без однобічності
- Отримання сформованої, колективної відповіді замість однієї думки
- Навчання розуміння того, як думають різні AI-системи
- Документування консенсусу для вашої команди

## Можливості

- **Мультиагентні дебати** — До 8 AI-агентів (Nemotron 120B, Llama 3.3 70B, Mistral Small 3.1, Gemma 3 27B — всі безкоштовні). 400+ додаткових моделей через OpenRouter
- **3-фазний двигун дебатів** — Opening (всі висловлюють думку) -> Discussion (@mentions, ліміт 5 на модель / 8 загалом) -> Summary (модератор підсумовує)
- **AI-модератор як учасник** — Модератор ділиться своєю думкою, оцінює аргументи інших, задає питання через @mentions та @ALL, підсумовує дебати. До 3 циклів модерації
- **@ALL тригер** — Звернення до всіх учасників одночасно. Всі хто вже висловився — відповідять
- **Автоматичний ретрай** — 3 повторних запити при помилці (5с, 15с, 30с) зі статусними повідомленнями в чаті
- **Прикріплення файлів** — Текстові файли (код, Markdown, CSV, JSON, до 100KB) включаються в контекст AI
- **Потокова передача** — Токен за токеном через SSE в реальному часі
- **@згадки** — Автодоповнення при введенні `@`, кольорове виділення, навігація клавіатурою
- **Температура** — Creative (0.9), Balanced (0.7), Precise (0.3)
- **Веб-пошук** — Іконка глобуса для режиму `:online` OpenRouter
- **Розширене мислення** — Підтримка chain-of-thought (згортувана секція)
- **Експорт в Markdown** — Заголовок з учасниками та модератором, часові мітки
- **Проактивна кнопка API Key** — Додайте свій ключ будь-коли для розблокування платних моделей
- **Сесії дебатів** — Збереження, завантаження, видалення. Автозбереження
- **Обробка збоїв** — Червоні індикатори, автоперепризначення модератора
- **Темна/світла тема**, регулювання шрифту, колапс повідомлень, мобільна адаптивність
- **Публічний режим** — Безкоштовні моделі без ключа; платні — з ключем користувача

## Як працює алгоритм дебатів

**Фаза 1: Ін'єкція теми**
Ви вводите питання або тему. Усі активні агенти отримують його як контекст.

**Фаза 2: Незалежні позиції (Раунд 1)**
Кожен агент формулює самостійну позицію, не знаючи про інших. Паралельна обробка гарантує справедливість.

**Фаза 3: Крос-запилення (Раунди 2+)**
Кожен агент тепер бачить відповіді інших. Генерується справжня реакція, контраргументи, визнання гарних точок.

**Фаза 4: Модерація (ваша роль)**
Ви втручаєтесь будь-коли:
- Уточнюйте терміни, якщо виникає плутанина
- Оскаржуйте слабкі аргументи з доказами
- Попросіть конкретного агента захистити позицію
- Модератор синтезує дебати й підсумовує консенсус

**Фаза 5: Конвергенція**
Через ітеративний обмін агенти:
- Уточнюють позиції на основі критики
- Визнають сильніші аргументи й коригують точку зору
- Рухаються до консенсусу (або виявляють непримиренні розходження, які варто знати)

```bash

```

```bash
git clone https://github.com/Lexus2016/Agent-Debate-Consensus.git
cd Agent-Debate-Consensus
npm install
cp .env.example .env.local   # Вкажіть ваш API-ключ
npm run dev
```

### Налаштування

Відредагуйте `.env.local`:

```env
OPENROUTER_API_KEY=your_key_here
MAX_ACTIVE_MODELS=8
```

Отримайте API-ключ на [openrouter.ai/keys](https://openrouter.ai/keys). Для **публічного режиму** залиште порожнім.

Відкрийте [http://localhost:3000](http://localhost:3000) в браузері.

### Оновлення

**Якщо встановлювали через Git Clone:**
```bash
git pull origin main
npm install
```

**Якщо встановлювали через NPX degit:**
```bash
npx degit Lexus2016/Agent-Debate-Consensus my-debate-updated
```
Потім перенесіть `.env.local` зі старої директорії в нову.

**Встановити конкретну версію:**
```bash
npx degit Lexus2016/Agent-Debate-Consensus#v2.1.0 my-debate
```

## Як користуватися платформою

### Початок дебатів

1. **Виберіть агентів** на лівій панелі. Кліком активуйте модельки, які хочете залучити до дискусії.
2. **Введіть питання або тему** в поле введення знизу. Це стане закріпленим контекстом для всіх агентів.
3. **Спостерігайте в реальному часі** — агенти видають свої позиції потоком, токен за токеном.

### Управління дебатами

- **@згадки для спрямування** — Напишіть `@Claude` чи `@Gemini` для адресації питання конкретній моделі.
- **Встановіть модератора** (натиснете зірочку ★ на агенту) — вибраний агент синтезує дебати й підсумовує позиції.
- **Втручайтеся будь-коли** — додавайте уточнення, оскаржуйте аргументи, попросіть розвинути ідею.
- **Кнопка розвинення** (⚡) — попросіть агента більш детально розглянути питання.

### Контроль розмови

- **Температура** — Creative (вільна творчість), Balanced (змішано), Precise (логіка).
- **Розмір шрифту** — Слайдер для комфорту читання.
- **Темна/світла тема** — Перемикач у шапці.
- **Копіювання** — Копіюйте повідомлення в Markdown формату.

### Збереження й експорт

- **Автозбереження** — Сесія зберігається в localStorage при перемиканні.
- **Експорт в Markdown** — Завантажте всю дискусію як .md файл для команди.
- **Перезавантаження** — Повернитеся до збереженої сесії з історією.

### API кінцеві точки

**POST `/api/chat`** — Stream SSE дебати
```

{
  "message": "Яка найкраща мова програмування?",
```

## Часті запитання

**Чи мій API-ключ безпечний?**
Ключ передається прямо від браузера до OpenRouter (публічний режим) або від сервера. Не реєструється, не логується.

**Скільки коштує запуск 8 агентів одночасно?**
Залежить від вибраних моделей. Kimi K2 та Grok 4.1 дешеві (~$0.001/1K токенів). Claude й Gemini дороговіші. OpenRouter розраховує по спожитих токенах.

**Можу я додати свою модель?**
Так. Будь-яка модель з OpenRouter доступна в каталозі. Новые моделі синхронізуються автоматично.

**Як змінити тему дебатів?**
Почніть новий чат (нова сесія). Перше повідомлення стає закріпленою темою для всієї дискусії.

**Що робити, якщо модератор дасть помилку?**
Система автоматично переназначає іншого агента модератором. Дебати продовжуються без перерви.

**Розміщення на власному сервері?**
Повна support. Деплой на Vercel, Netlify, власний сервер — будь-де. Вам потрібні тільки Node.js 16+ і OpenRouter ключ.
